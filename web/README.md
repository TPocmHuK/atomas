# Atomas — HTML5-клон под Яндекс Игры

Чистый веб-клон Atomas: HTML + CSS + Canvas, без сборщиков, без зависимостей.
Подключён [Yandex Games SDK v2](https://yandex.ru/dev/games/doc/ru/sdk/sdk-about): межстраничные ролики, реклама за награду (продолжение игры), лидерборд и облачное сохранение рекорда.

## Локальный запуск

Открыть `index.html` через файловый протокол **не получится** — SDK подгружается с CDN, плюс нужен валидный `origin`. Запустите любой статический сервер:

```bash
cd web
python3 -m http.server 8080
# или
npx serve .
```

И откройте `http://localhost:8080`. В локальной среде SDK молча упадёт (ничего страшного), а игра будет работать без рекламы и без облачного рекорда (рекорд сохранится в `localStorage`).

## Сборка архива для Яндекс.Игр

Яндекс Игры ожидают ZIP, в **корне** которого лежит `index.html`. Перейдите в папку `web/` и упакуйте её содержимое:

```bash
cd web
zip -r ../atomas-yandex.zip . -x "README.md" "*.DS_Store"
```

Готовый `atomas-yandex.zip` загружается в [консоль разработчика](https://yandex.ru/dev/games/console).

## Что нужно настроить в консоли Яндекс.Игр

1. **Идентификатор игры** — после загрузки архива консоль выдаёт `app-id`. Никаких правок в код вносить не нужно, SDK сам подхватит окружение.
2. **Лидерборд** — создайте лидерборд с технологическим именем `main` (используется в `submitScore('main', best)`). Если используете другое имя — поменяйте константу в `js/main.js`.
3. **Реклама** — типы рекламы (`fullscreen`, `rewarded-video`) включаются в настройках игры. До прохождения модерации SDK возвращает тестовые объявления.
4. **Локализации** — игра поддерживает `ru`, `en`, `tr` (`js/i18n.js`). Добавляйте новые языки прямо туда же.

## Структура

```
web/
├── index.html       # разметка + загрузка YSDK
├── style.css
└── js/
    ├── atoms.js     # символы и палитра элементов
    ├── game.js      # чистая игровая логика (ring + plus-fusion)
    ├── render.js    # рисование на Canvas + хит-тестинг
    ├── i18n.js      # ru/en/tr
    ├── ysdk.js      # обёртка над Yandex Games SDK
    └── main.js      # склейка DOM ↔ state ↔ renderer ↔ SDK
```

## Особенности интеграции SDK

- `YSDK.init()` сообщает о готовности через `features.LoadingAPI.ready()` — спиннер Яндекса исчезнет, когда игра реально готова.
- `YSDK.ready()` / `YSDK.gameplayStop()` дёргают `GameplayAPI`, чтобы Яндекс не показывал рекламу прямо посреди игры.
- Полноэкранная реклама троттлится самим SDK (не чаще 60 сек) — мы дополнительно ограничиваем «не чаще одного раза в 25 ходов».
- Рекламу за награду показываем по кнопке «Продолжить» на экране поражения; после показа из кольца удаляется половина атомов с наименьшим номером.
- Рекорд кладём в `localStorage` **и** в `player.setData({best})` (если SDK доступен и игрок авторизован).
